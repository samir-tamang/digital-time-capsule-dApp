# Digital Time Capsule

A decentralized app that encrypts messages, locks them on Ethereum until a set date, and stores them permanently on IPFS.

##  Features
AES-256-GCM Encryption(client-side only)
Blockchain Time-Lock (enforced by smart contract)
Decentralized Storage (IPFS via Pinata)
MetaMask Integration
Fully Responsive(mobile & desktop)

## Tech Stack
frontend**: Next.js 14, React, TypeScript, Tailwind CSS v4
Blockchain**: Ethereum Sepolia, Hardhat, ethers.js v6
Storage**: IPFS (Pinata)
Crypto**: Web Crypto API
